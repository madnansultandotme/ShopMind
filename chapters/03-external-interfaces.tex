\section{External Interface Requirements}
\subsection{User Interfaces}
\begin{itemize}
    \item Responsive web interface for inventory, billing, reporting, and assistant chat.
    \item Role-based navigation menus and dashboards.
    \item Search and filter controls for products, invoices, and sales.
    \item Visual cues for stock alerts and key insights.
\end{itemize}

\subsection{Hardware Interfaces}
No special hardware is required beyond standard POS peripherals (optional): barcode scanner and receipt printer.

\subsection{Software Interfaces}
\begin{itemize}
    \item PostgreSQL for data storage.
    \item OpenAI API for LLM processing.
    \item REST API for frontend-backend communication.
\end{itemize}

\subsection{Communications Interfaces}
All client-server communication shall use HTTPS. The system shall use JSON over REST for API calls.

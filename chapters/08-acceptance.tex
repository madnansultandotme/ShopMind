\section{Acceptance Criteria}
\begin{itemize}
    \item Users can add a product and see it in inventory within 2 seconds.
    \item A cashier can generate an invoice that updates stock correctly.
    \item A manager can ask a natural-language question and receive a correct answer backed by data.
    \item Daily sales reports can be generated and exported.
    \item Audit logs show the correct user and timestamp for inventory edits.
\end{itemize}
